This Week in Genesis (27 Aug 2017)

27 August 2017 by Nick Davis


Photo by Braden Collum on Unsplash

This Week in Genesis, a Genesis starter theme with CSS Grid support, updates to the Genesis Simple Share plugin, and your opportunity to guest edit This Week in Genesis.

Themes

One for the devs amongst you, Sridhar Katakam launches v2.3.0 of his Genesis Starter theme (based on v2.3.0 of the popular Genesis Sample theme).

It features Theme Logo support, inline navigation menu, CSS Grid support for structural elements, mobile first CSS and more.

Genesis Starter 2.3.0:https://t.co/9kw7vsKOWd#GenesisWP

— Sridhar Katakam (@srikat) August 23, 2017

Plus, if you’re interested in discussing future improvements to the Genesis Sample theme, there’s a new channel (#sample) in Slack just for you.

We have a new channel in GenesisWP Slack named #sample for those interested in improving Genesis Sample w/ CSS Grid/Flexbox etc.

— Sridhar Katakam (@srikat) August 22, 2017
