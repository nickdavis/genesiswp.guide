WordPress 4.7 and Genesis

WordPress 4.7 was released this week, but before you go running to update make sure you’ve updated to the latest version of Genesis (or at least v2.4.0).

PSA: #genesiswp users, you need to be on at least Genesis 2.4.0 (released 9/28) for WordPress 4.7, which is scheduled for 12/6.

— Robin Cornett (@robincornett) December 5, 2016
