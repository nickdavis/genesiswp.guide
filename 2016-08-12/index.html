And finally…

Just a reminder, as WordPress 4.6 draws closer, if you didn’t already you should update all of your WordPress websites to Genesis 2.3.1 and ask your clients to do the same.

By not doing so you risk an error on those sites as soon as WordPress updates to 4.6, so don’t delay!

This is a Must Read if you haven't updated to the latest version of #GenesisWP. @lindseyriel https://t.co/iiq6V8C637

— Susan Ramsey (@onehappystudio) August 11, 2016
