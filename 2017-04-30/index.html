Genesis 2.5

Genesis lead developer, Nathan Rice, reveals that Genesis 2.5 is now available.

Nathan describes 2.5 as making:

significant progress, both in conversation and code, in starting to move Genesis into a better-organized, more object-oriented direction.

Highlights include:

  • breaking up larger blocks of HTML into separate views
  • further improvements to the Markup API
  • extracting more of the legacy XHTML logic to a separate file
  • full support for the new WordPress title tag

For the full changelog and more details, check out Nathan’s blog post.
